## How to Use

1. Clone the repository.
2. Open the folder in VS Code, Jupyter Notebook, or any Python IDE.
3. Run Python scripts from the relevant week/day folder.
4. Open `.ipynb` files in Jupyter Notebook, JupyterLab, or VS Code.

Example:

```bash
python "Python Codes/Week 1/Day 1/arithmetic.py"
```

## Requirements

Most beginner Python scripts only require Python. For notebooks and data analysis tasks, install the common data science libraries:

```bash
pip install numpy pandas matplotlib jupyter
```
